Ethercrypt

Secure password manager & TOTP authenticator

About

Ethercrypt is a lightweight, zero-knowledge password manager designed to securely store credentials and generate TOTP codes across multiple platforms.

All data is encrypted locally before storage. The developer cannot access user vault data.

Storage Options

  • Local encrypted storage (device-only)
  • Google Cloud Firestore (user-configured)
  • Google Drive (planned)
  • OneDrive (planned)
  • Dropbox (planned)

Security Model

  • Zero-knowledge architecture
  • Client-side encryption
  • No analytics or telemetry
  • No central server for vault data
